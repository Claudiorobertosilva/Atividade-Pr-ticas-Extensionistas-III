# 🏥 Sistema de Gestão de Consultas de Saúde

Este projeto tem como objetivo desenvolver a modelagem de um sistema de informação voltado para o gerenciamento de consultas médicas, incluindo a estrutura de banco de dados e diagramas UML.

---

## 📌 📖 Descrição

O sistema permite o controle de agendamentos médicos, relacionando pacientes, médicos e clínicas. A modelagem foi desenvolvida com base em boas práticas de Engenharia de Software, utilizando:

* Modelagem Entidade-Relacionamento (DER)
* Diagramas UML
* Estrutura de banco de dados relacional

---

## 🎯 🎯 Objetivos

* Modelar um sistema de consultas médicas
* Criar um banco de dados consistente
* Representar o sistema com diagramas UML
* Facilitar o entendimento do funcionamento do sistema

---

## 🧩 📊 Diagramas Desenvolvidos

* ✔ DER Conceitual
* ✔ DER Lógico
* ✔ Diagrama de Classes
* ✔ Diagrama de Caso de Uso
* ✔ Diagrama de Sequência
* ✔ Diagramas de Atividades

---

## 🏗️ 🗄️ Estrutura do Sistema

O sistema é composto pelas seguintes entidades principais:

* **Paciente**
* **Médico**
* **Clínica**
* **Consulta**

A entidade **Consulta** é responsável por relacionar as demais.

---

## 🔗 🔗 Relacionamentos

* Um paciente pode realizar várias consultas (1:N)
* Um médico pode atender várias consultas (1:N)
* Uma clínica pode registrar várias consultas (1:N)
